HYPERLINKS

A hyperlink is a word, phrase, or image that you can click on to jump to a new document or a new section within the current document. Hyperlinks are found in nearly all Web pages, allowing users to click their way from page to page.

Absolute link

An absolute link is a hyperlink containing a full URL, which includes all the information needed to find a particular site, page or document or other addressable item on the Internet.

Relative link

A relative link is any link that shows the current URL's relation to the linked page's URL. Instead of having the whole reference URL in the h-ref tag, relative links only show the relative link paths, as their name suggests. It's the difference between a link that has the website's full address, and a link that only includes a file's specific location.

Internal link

An internal link is a link from one page to another page on the same domain. They're simply text hyperlinks from one page to another page on your website.

embedded kinks

A little tougher to define, embedded links are just another way of saying a link that when clicked, leads somewhere else. Embedded links can be more than text though. You can embed an image as a link to another page on the web.
